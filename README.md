自动驾驶,
其实就是可以自动处理意料中的情况,
只是这情况是一件事儿,随机发生,
还是无数件事儿,随机同时发生,
从处理难度上当然不同,

可在自动驾驶角度看是一样的:
能自动处理,

就算自动驾驶.

有限的已知情景变化,
是自动驾驶;

有效处置一切突发场景,
