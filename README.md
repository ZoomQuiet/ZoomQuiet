什么是超临界?
> ...水的临界参数:22.129MPa/374.15℃;
在这个压力和温度时水和蒸汽的密度是相同的:即 水的临界点;
能够达到甚至超过这个临界点,
煤的发电效能就能够提升;
超临界(Supercritical)燃煤发电是指:

+ 容量为60万千瓦以上
+ 主蒸汽压力达到25兆帕以上
+ 温度达到摄氏593度以上
+ 机组热效率能够达到45% ...

是的,即便是高科技的超临界率能也没过半,

也就是说每燃烧一吨煤,
超过一半浪费热量被浪费,

转换为其它不可利用的能量/物质扩散到大气中;

所以,
