    0: 穷, 只能将后代当成劳动力来储备, 多生娃, 多开荒, 多打粮
    1: 富, 必须多生娃, 多分封, 才能多占地, 内部自由竞争, 长久发展

- 需要高质量人口来持续提升生产效能
