+ 就是说张三丰五十岁时灭绝出生, 灭绝在张三丰六十五岁时接任掌门之位, 时年十五岁;
+ 就是说风陵师太于张三丰六十五岁时去世;

+ 郭襄圆寂后把掌门之位传给了风陵师太
+ 风陵师太当了掌门后第三年圆寂, 把掌门之位传给了年仅十五岁的灭绝师太;
