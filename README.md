无论手机中展现的是什么内容/数据,

另外的时空,

    i'am the God

不象所在的时空, 几乎一切都受到各种制约,
当然,
理性思考一下就知道,
手机中的时空受到的控制更加精密, 更加不受自己控制,

却是所有人都需要的...

所以, 
忍不住拿出手机来刷一刷,
并不是为了手机本身, 或是手机带来的一切,
而是可以借机`假装跳出当前无聊的自己`...
