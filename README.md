> 是什么让你频繁地不由自主拿出手机？

跳出当前无聊的时空,

    i'am the God

当然,
理性思考一下就知道,
手机中的时空受到的控制更加精密, 更加不受自己控制,

却是所有人都需要的...

而是可以借机`假装跳出当前无聊的自己`...
