174年前的今天, "共产党宣言"正式出版, 之后被翻译为几十种语言风行全球, 给所有民族带去又一种选择; 其中汉语版本前后得有上百种译本, 不过, 中央编译局的, 最为用心, 甚至于文中很多早已消失的职业, 建国后, 为了翻译精当还专人调研后, 重新确立译名...
