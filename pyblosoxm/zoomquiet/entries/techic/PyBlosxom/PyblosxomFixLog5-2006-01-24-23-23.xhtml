3. 教训

MainlyGreen
此样式，的排版非常古怪，折腾了半天，越调越乱； 最后还是调出自个儿donews 的样式 , 使用 69%-25% 的百分比来组织版块。
教训:
  1. CSS 和代码开发一样，没有理解就不能尝试
  2. CSS 的增长比代码还要快,所以最后应该进行精简
  3. CSS2 对IE的兼容性，就不要折腾了，先调试FireFox 再 Opera,最后看IE 再不成，就直接禁止IE访问吧…………
